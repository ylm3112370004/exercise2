# Readme

### 步骤

* Fork 代码仓库并拉到本地
* 安装依赖

```
npm install
```
* 启动测试

```
npm run test
```

* 打开 `./lib/add.js` 修改代码跑通测试用例 
* 提交github，尝试接入Github Actions

### 参考

* [Jest](https://jestjs.io/zh-Hans/)
* [Github Actions](https://docs.github.com/cn/actions)
